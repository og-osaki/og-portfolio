# OG Portfolio

## 概要
これまでのHTML、CSS学習のまとめとして作成したポートフォリオサイトです。ごく初歩的なJavaScriptの使用も試みましたが、主なコンテンツとして、Blenderで作成した3Dモデルを掲載しています。

## URL
※現時点ではローカル環境での確認を想定しています。

## 使用技術
- HTML5
- CSS3
- JavaScript (ES6)

## 制作方針
作者の現時点でのプログラミングスキルの目安となるようなサイトを目指しました。そのため、原理や構文の理解が十分でない要素は採用しないよう心掛けました。

## 工夫した点
- セマンティックHTMLを意識した構造
- レスポンシブ対応（PC、モバイル）
- id・クラス立ての統一性、コード自体の可読性
- 将来のコンテンツ追加のしやすさ

## 苦労した点
- margin / paddingやflexの性質を見直し余白設計のムダ・重複をなくすこと
- 実現したい挙動をできる限り現在のスキルの範囲内で実装すること

## 今後の挑戦
- 3Dモデルや、観光地で撮影した360°画像をVRビューワなどで埋め込みたい
- モバイル表示時にハンバーガーメニューを使いたい
- JavaScriptのより高度な活用
